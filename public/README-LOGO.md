# Logo OLIMPIKA FITNESS

Para exibir a logo no sistema (tela de login e demais telas):

1. Coloque o arquivo da logo em: **olimpika-logo.png** (nesta pasta `public/`).
2. A logo é exibida **sem alteração** de cor ou design — apenas redimensionada para enquadrar (proporção mantida).
3. Na tela de login a logo aparece em destaque no topo; nas outras telas, no tamanho adequado.

Se o arquivo não existir, o app exibe o texto "Olimpika Fitness" como fallback.
